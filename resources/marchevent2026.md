# 💻 Learning Programming for Cognitive Science

📘 **Programming in Psychological Science – A Practical Introduction**  
A practical and accessible book introducing programming for psychology students and researchers.[Here](https://wobc.github.io/programming_book/)


👩‍💻 **SheCodesPsy – 3-day Programming Course**  
An immersive course designed for female+ psychology *students* who want to learn programming.[Here](https://wobc.github.io/programming_book/](https://shecodespsy.github.io/)

# 🤝 Contributing to Open Science

🌍 **The Turing Way – A Handbook for Reproducible, Ethical and Collaborative Data Science**

Learn how to contribute to open science and open-source projects.[Here](https://book.the-turing-way.org/)

🧪 **PsychoPy GitHub Repository**

Explore or contribute to the open-source tools used widely in psychology and neuroscience experiments.[Here](https://github.com/psychopy/psychopy)
